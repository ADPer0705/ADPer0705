## 🔭 What I'm Building

| Project | Description |
|---|---|
| [**ChronoTrace**](https://github.com/GR4Y-SCALE/ChronoTrace) | NTFS anti-forensics detection framework — detects timestamp manipulation and log evasion via post-mortem image analysis and live monitoring |
| [**Inquiro**](https://github.com/CY-N4P5E/Inquiro) | Local-first RAG system over private document collections using LangChain, FAISS, and Mistral via Ollama |
| [**MerkleWatch**](https://github.com/ADPer0705/MerkleWatch) | CLI utility for tamper-evident directory snapshots using Merkle trees and SHA-256 |
| [**Praetor**](https://github.com/ADPer0705/praetor) | Serverless AWS compliance platform — 170+ daily security checks across 11 services, zero cost |
| [**Wordsmith**](https://adper.me/wordsmith) | AI-driven contextual wordlist generator for security professionals, powered by multiple LLM APIs |
